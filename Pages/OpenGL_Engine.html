OpenGL Skinning Engine

Graduate Project

OpenGL engine project I worked over a year, adding new features and systems. We also developed a converter project along with it to convert the .glb models into custom protobuffer files to import into the engine.



  • In-engine primitive shapes in OpenGL.
  • Multiple shader types included simple lit
  • .GLB 3d-models to protobuffer file cmd-line tool
  • Collision sphere radius calculation
  • Animated game objects
  • Animation controls
  • 2D Sprite and UI support
  • Multiple Cameras support
  • Don't mind the broken bot
  • Skinning on Compute shaders
  • Animation blending on Compute shaders
  • Lossless Animation data compression
  • Haunted mousey
